# Tasks Marketing Video

## Summary

This storyboard sells the redesigned Tasks page as a calm, family-friendly planning surface. It starts from Home, lands on the new Tasks layout, then shows fast scanning, progressive disclosure, completion, tomorrow planning, and a quiet hero-shot ending.

## Scenes

1. Home — show today’s tasks on Home, then navigate into Tasks.
2. First Impression — hold on the redesigned Tasks layout so the new hierarchy can be absorbed.
3. Scan Speed — move the cursor slowly across the columns without clicking.
4. Progressive Disclosure — select tasks to reveal actions, then click away to hide them again.
5. Complete Task — complete a task and let the movement finish before revealing the result.
6. Move to Tomorrow — move a task into Tomorrow and wait for the visible shift.
7. Planning — briefly interact with Tomorrow and This Week to show planning stays simple.
8. Overall Dashboard — end on a calm, idle hero shot.

## Key UX Messages

- Today-first workflow.
- Reduced visual noise.
- Progressive disclosure.
- Improved scan speed.
- Better visual hierarchy.
- Better desktop layout.
