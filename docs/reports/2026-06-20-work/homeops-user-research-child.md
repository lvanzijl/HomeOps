# HomeOps User Research — Child Persona

## Persona

I am 8 years old. I can read. I like rewards, visible progress, stars, and seeing things marked done. I do not want complicated setup.

## How I understood HomeOps from the guide

HomeOps is a family board. I can see my family, my avatar, tasks, and progress. Parents add most things. I want to know what I should do and what I get when we finish.

## Initial setup from an empty installation

### Profiles / household members

I expected to see my own name. I cannot create the family myself. If my name is not already there, I need a parent to set it up. The guide says family members are not login accounts.

**Result:** I do not really have my own account. I am a family member shown on the board.

### Avatars

I like changing my avatar. I can choose child/adult, hair, glasses, skin tone, shirt color, and display color. Some options use color codes that I do not understand, but I can still choose.

**Result:** This is fun and easy enough with help.

### Goals

I wanted to choose a goal or see a reward. The Motivation page shows goals, stars/checkmarks, and family progress, but I cannot set my own goal. A parent also cannot prove they can set it from the user interface.

**Result:** I can look at goals, but I cannot make them mine.

### Tasks

Parents can create tasks for me. I can see task names, due dates, and owner names. I can press Complete when I finish. If a task was marked done by mistake, it can be reopened.

**Result:** I understand tasks if the list is short.

### Shopping lists

Shopping is mostly for parents. I could add snacks if allowed, but it is not a child activity.

**Result:** Not very interesting to me.

### Calendars

Calendar events tell me what is happening, like swimming lesson. I would not add many calendar events myself.

**Result:** Useful if parents keep it updated.

### Rewards and motivation

The stars and progress bar are the most fun part. I want the reward to be clear and I want progress to change when I complete tasks. The guide says task completion can update progress.

**Result:** Fun idea, but it needs configurable rewards, badges, and clearer achievements.

## Several weeks of hypothetical use

### Week 1

I would open my avatar page first. I would want to make it look like me. Then I would ask what tasks I need to do. I would press Complete after finishing tasks.

### Week 2

I would check Motivation to see stars. If my tasks fill stars, I would keep checking. If the goal does not change or the reward is not something I care about, I would stop caring.

### Week 3

I would only use HomeOps if a parent reminded me and if progress felt real. I would not use Settings, Lists, House Status, Media, or calendar administration. Placeholder pages would be boring.

## Evaluation by topic

### Discoverability

Good: family avatars on Home are easy to find. Motivation sounds interesting. Complete buttons on tasks are understandable.

Weak: I do not know what to do next without a parent. There is no child home page, no “my tasks” page, and no obvious reward shop.

### Onboarding

I need a parent-led setup. There is no kid-friendly first-run explanation.

### Information architecture

Home and Motivation make sense. Tasks are okay if not too many. Settings and placeholders are not for me.

### Feature completeness

Good: avatar, task completion, progress display.

Missing for me: my own task list, rewards I can choose, badges, levels, coins/gems, celebration after progress, and clear child-friendly goals.

### Household setup workflow

I cannot set up the household. I depend on parents. If parents cannot configure goals, my motivation features are weak.

### Conceptual UX

I like the idea of doing tasks and seeing stars. I want the system to feel like progress, not just chores.

### Long-term usefulness

I would keep using it only if parents keep tasks updated and the Motivation page gives rewards I care about.

## Adoption assessment

**Would likely abandon unless parents actively drive it.** The visible progress is promising, but there are not enough fun or configurable reward features for me.
