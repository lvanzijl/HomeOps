# Persona Simulation: Child Age 6

## Persona
Reads a little, likes stickers, limited patience.

## Reading the Guide
A six-year-old cannot independently use the full guide. They understand avatars, stars/checkmarks, and simple family progress only with adult help.

## Empty Installation Walkthrough

| Step | What they try | Success? | Struggle / understanding |
|---|---|---|---|
| Configure household | Parent-led | No independent success | Too much reading/admin. |
| Add family members | Watches parent add them | Partial | Recognizes their name/avatar afterward. |
| Configure avatars | Chooses visual options with parent | Yes with help | This is engaging and understandable. |
| Configure goals | Looks for their goal | Partial | Visual progress helps, but goal creation is adult-oriented. |
| Configure rewards | Expects stickers/prizes | No | No rewards/redemption; likely disappointment. |
| Configure tasks | Parent assigns tasks | Partial | Can understand “Complete” with help, but no child-specific simplified view is proven. |
| Configure shopping lists | Not primary user | No | Too text-heavy and adult-oriented. |
| Configure calendars | Not primary user | No | Calendar workflow is adult-oriented. |
| Multiple hypothetical weeks | Checks avatar/progress | Weak | Engagement may fade without stickers, rewards, celebrations, or simplified child mode. |

## Assessment
For age 6, child engagement is **Weak**. Avatars and stars help, but the product lacks the proven sticker/reward loop and simplified child-specific task view needed for independent use.
