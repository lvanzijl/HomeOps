# Home Task Visual Integration

## Summary
Integrated Tasks as the third Home household pillar while preserving the glassboard hierarchy.

## Notes
- Agenda remains the largest Home summary column.
- Shopping/Lists remains visible as the memory surface.
- Tasks uses the existing Tasks teal domain accent and a compact row cap so it does not dominate Home.
- If task volume grows, Home shows fewer rows plus overflow instead of expanding into task management.
