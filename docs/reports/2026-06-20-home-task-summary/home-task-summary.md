# Home Task Summary

## Summary
Added a bounded Tasks summary card to Home so the dashboard now surfaces overdue, due today, and upcoming household responsibilities.

## Decisions
- Home shows task awareness only: title, owner/shared household indicator, and due date.
- Completed tasks and no-due-date tasks stay off Home.
- Recurrence, approval, points, categories, notifications, reminders, and task configuration remain absent from Home.
- The summary caps visible rows and uses overflow to route to Tasks rather than expanding Home.
